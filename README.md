## 项目简介

在Opengl的规范下，使用glfw和glad先完成了对三角形的绘制、纹理的填充等基础图形渲染功能。除此之外，我还完成了一个小的项目，可以通过鼠标或者键盘事件来控制摄像机的视角和位置的变化，模拟类似CAD和第一人称游戏视角控制摄像头观察世界。
